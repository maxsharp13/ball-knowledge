# Ball Knowledge

Ball Knowledge is a full-stack basketball learning platform where users can learn basketball terminology, upload custom plays, and explore community-created concepts.

## Features

- User authentication
- Basketball concepts and formations
- Community uploaded plays
- Protected routes
- MongoDB database integration
- Responsive React frontend

## Technologies

- React
- React Router
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JWT Authentication


## Backend

Express + MongoDB Atlas backend with protected API routes.
